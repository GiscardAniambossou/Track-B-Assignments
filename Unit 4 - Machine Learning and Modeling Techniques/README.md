# Unit 4: Machine Learning and Modeling Techniques

Students will learn how to build and implement machine learning models that can help their organization answer big data questions.  
Students will learn some of the most popular and used algorithms for supervised and unsupervised machine learning.

## Some topics this unit will cover following machine learning algorithms: 
+ Linear regression
+ Scikit-Learn

## Outcomes of this unit:
+ A solid foundation of machine learning techniques, and how to use them in Python.
+ Understand and use the modeling process and why it’s important for all machine learning. 
+ How to leverage Python’s advanced libraries to perform Machine Learning.
